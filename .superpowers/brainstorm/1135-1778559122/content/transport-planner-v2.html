Filter row breakdown

Dropdowns (left group)

Leave, Arrive, Price, Sort — 4 compact pill-dropdowns in a row. Labels shortened to fit. Active filter highlighted in Ember. Open on click, close on blur or selection.

Divider

A 1px vertical rule separates the dropdown filters from the toggle pills — two distinct input types, visually grouped.

Toggle pills (right group)

Direct, Eco, Flexible, Cabin bag — boolean toggles. Active = Ember fill + border. Inactive = muted. Click to toggle on/off. Results filter live.

Overflow

Row scrolls horizontally on narrow screens — no wrapping, no second line. Single row always.

Mode-specific pills

Drive mode swaps pills for: Tolls, Fuel type, EV only. Ferry: Cabin class, Vehicle. Train: Rail pass, 1st class. Same row, different set.
